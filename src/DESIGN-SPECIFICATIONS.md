Brand Guidelines

Primary Colors

- Main Brand Color: #3B2322 (Coffee Brown)
- Secondary Colors: White, Gray Scale (50-900)
- Accent Colors: Amber-100, Green-500 (for verification)

Typography

- Primary Font: Inter (via Google Fonts)
- Font Weights: 300-900 available
- Heading Sizes:
  - H1: 4xl-5xl (Mobile-Desktop)
  - H2: 3xl
  - H3: 2xl
  - Body: base/text-lg

Spacing System

- Container Padding: px-4 (mobile), auto-centered
- Section Padding: py-12 (standard), py-20 (hero)
- Gutters: gap-8 (sections), gap-4 (elements)

Responsive Breakpoints

-Mobile: < 768px (flex-col, full-width)
-Tablet: 768px - 1024px (grid-cols-2)
-Desktop: > 1024px (grid-cols-3/4, side navigation)

Design Principles

1. Minimalist: Clean lines, smple white space
2. High Contrast: Dark brown on white background
3. Consistent: Uniform spacing and sizing
4. Purposeful: Every element serves a clear function
5. Sister Site: Complementary to AcaciaBeverage.com

Layout Grid

All pages follow:

1. Container: max-w-7xl mx-auto px-4
2. Section: py-12 with clear heading hierarchy
3. Content: Organized in responsive grid (1-2-3-4 columns)
4. Footer: Consistent across all pages

Mobile-First Considerations

- Navigation: Hamburger menu on mobile
- Cards: Stack vertically on mobile
- Text: Reduce font sizes on mobile
- Padding: Reduce section padding on mobile
